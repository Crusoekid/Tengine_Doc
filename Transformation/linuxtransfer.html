编译Linux转换工具

总共有3种转换方式：

  • 下载已经编译好的编译工具tm_convert_tool，在Linux上运行。

  • 网站在线转换。

  • 编译源码进行转换

    git clone https://github.com/OAID/Tengine-Convert-Tools.git
mkdir build
cd build
cmake .. && make -j4
